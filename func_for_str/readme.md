Здесь представлены собственные реализации функций из string.h такие как:

strdup, strtok, strlen, strchr, а так же функция get_str() для чтения строки

Программа в файле main демонстрирует работу этих функций.
Она принимает предложение на английском и к каждому слову, которое начинается с согласной буквы, добавляет префикс
