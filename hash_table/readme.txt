 Хеш-таблица с открытой адресацией на основе вектора, использующая перемешивание сложением(линейное пробирование)
 • ключ —беззнаковое целое число;
 • значение — нуль-терминированная строка произвольной длины

Реализовованы основные функции таблицы:инициализация, вставка, удаление, поиск, очситка таблциы, вывод на консоль.
Также реализованы методы сохранения таблицы в бинарный файл и загрузки оттуда.
